Introduction

This website was created as part of my web development assignment. The aim was to design and build a multi-page portfolio website using HTML and CSS. The website includes a few pages such as a homepage, project page, contact page, video demonstration page and this report page.

Design and Layout

The website uses a consistent layout across all pages, including a header, navigation bar, main content area, sidebar and footer. I used CSS Grid to structure the layout and ensure it remains organised and easy to read. The colour theme I picked was purple, thats because that's my favourite colour, and I have to chose to make it faint to black towards the bottom of the page, to give it a little more interesting colour scheme than a plain background. I think the purple is very nice and soft towards the enctype and does not need another colour in combination, as it would have been too much.

Development Process

I created the website step by step, starting with the basic structure of each page using HTML, and then added CSS to style the website and lay it out in a way that fits both assigment criteria and my personal preference. I used Visual Studio Code to write the code and GitHub to manage and save my progress. I made multiple commits to track changes throughout the development process, which can also help me go back if mistakes would have been made, which actually has happened at the beginning of the assigment for me, and git was very useful, as I was able to easily revert to my previous commit.

Features

The website includes a navigation menu that links all pages together which can be found at the top of the screen, in a floating island in the middle. It is responsive and adjusts to different screen sizes using media queries. The project page uses a grid layout to display content, and the contact page includes a form that allows users to send a message. The website also includes a video demonstration page, which is a video of me going through both the website and the code of the website.

Validation

The HTML and CSS were tested using validation tools to ensure there are no errors. This helps improve the quality of the code and ensures that the website follows correct web standards.

Reflection

During this project, I learned how to build a complete website using HTML and CSS. I improved my understanding of layout, styling, and responsive design. If I had more time, I would improve the design further and add more interactive features and more animations to the website.
